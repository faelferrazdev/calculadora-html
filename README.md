# Calculadora Web

Este projeto consiste em uma calculadora simples desenvolvida utilizando **HTML5, CSS3 e JavaScript**.
O objetivo do projeto é demonstrar conceitos básicos de lógica de programação, manipulação do DOM e interação com o usuário.

## 📷 Preview do Projeto

![Preview da Calculadora](calculadora-image.png)

## 🚀 Funcionalidades

* Realizar operações matemáticas básicas
* Interface simples e intuitiva
* Atualização dinâmica dos valores na tela
* Implementação utilizando JavaScript puro

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
